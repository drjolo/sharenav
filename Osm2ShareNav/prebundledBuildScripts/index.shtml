Pre-bundled maps for ShareNav

This page offers ready to use bundles of ShareNav including map data of a set of regions and countries. We try and offer a good selection of regions to make using ShareNav as easy as possible, so please do drop us a line if you would like to see other regions here, although unfortunately, as we have limited resources, we can't promise to include it. However, it should be easy enough to build your own midlets with exactly the region you want with the help of Osm2ShareNav.

These midlets are updated about once a month to contain the latest data from OpenStreetMap.

To install ShareNav on your phone, simply copy the .jar (and potentially the .jad file depending on your phone) to your device using for example either bluetooth or a usb connection after which you can select the file for installation.

If you are experiencing annoying permission questions or problems, you could try using the signed midlets in combination with the zip files offered here as external maps.

Some of the country extracts can be quite large 50 - 150Mb. Not all phones will be able to handle such large midlets. If you are seeing (performance) troubles, please try one of the smaller midlets.

Should you have any comments, suggestions or problems with these midlets, please let us know on the project site.
